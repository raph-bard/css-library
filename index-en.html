CSS Library Documentation

Installation:

To start using the CSS library, include the main.css file in your project before your own CSS file:

<link rel="stylesheet" href="path/to/main.css">

Basic Structure:

The library uses classes to style HTML elements. Here are some basic classes you can use:

  • Text-align: .text-center, .text-left, .text-right, etc..
  • Font-size: .text-1 to .text-12.
  • Line-height: .line-1 to .line-12
  • Font-weight: .font-1 to .font-9.
  • Text-transform: .text-uppercase, .text-lowercase, .text-capitalize.
  • Displays: .d-block, .d-inline, .d-none, etc.
  • Margins/paddings: .m-, .mt-, .mr-, .mb-, .ml-, .my-, .mx-, .p-, .pt-, .pr-, .pb-, .pl-, .py-, .px-.

Grid:

The library also provides a simple grid system for organizing your layouts. Here are the available grid classes:

  • .container: Sets a maximum width for the content and centers it horizontally.
  • .row: Defines a row in the grid.
  • Column classes: .col-1 to .col-12.

Customization:

You can also customize the library by adding your own styles or replacing default values of CSS variables. Some features, such as the scrollbar or the grid system, are customizable via variables in the corresponding SASS files.

Usage Examples:

Here are some usage examples:

Center an Element and Apply Bottom Margin:

Create a Simple Responsive Grid:

<div class="row">
