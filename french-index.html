Documentation de la bibliothèque CSS

Installation:

Pour commencer à utiliser la bibliothèque CSS, incluez le fichier main.css dans votre projet avant votre propre fichier css:

<link rel="stylesheet" href="path/to/main.css">

Structure de base:

La bibliothèque utilise des classes pour définir le style des éléments HTML. Voici quelques classes de base que vous pouvez utiliser :

  • Text-align: .text-center, .text-left, .text-right, .text-justify
  • Font-size: .text-1 à .text-12.
  • Line-height: .line-1 à .line-12
  • Font-weight: .font-1 à .font-9.
  • Text-transform: .text-uppercase, .text-lowercase, .text-capitalize.
  • Displays: .d-block, .d-inline, .d-none, etc.
  • Margins/paddings: .m-, .mt-, .mr-, .mb-, .ml-, .my-, .mx-, .p-, .pt-, .pr-, .pb-, .pl-, .py-, .px-.
  • Positions: .p-relative, .p-absolute, .p-fixed, .p-static, .p-sticky
  • Bordure: .bordered (Applique une bordure noire d'1px.)

Arrondis (Border-radius):

Pour arrondir les coins des éléments, utilisez les classes suivantes :

  • .rounded : Applique un border-radius de 50% pour créer un cercle.
  • .rounded-none : Supprime le border-radius.
  • .rounded-1 à .rounded-12 : Définit le border-radius avec différentes valeurs.

Flex:

Les classes flex permettent de contrôler l'alignement et le positionnement des éléments :

  • Horizontal Alignment: .align-center, .align-baseline, etc..
  • Vertical Alignment: .justify-center, .justify-between, etc..
  • Directions: .flex-row, .flex-column, etc..
  • Flex Wrap: .flex-wrap, .flex-nowrap, .flex-wrap-reverse
  • Gaps: .gap-1 à .gap-12.
  • Align Items: .align-start, .align-end, .align-center, etc..
  • Align Self: .align-self-start, .align-self-end, .align-self-center, etc..
  • Align Content: .content-start, .content-end, .content-center, etc..
  • Justify Content: .justify-start, .justify-end, .justify-center, etc..
  • Justify Self: .justify-self-start, .justify-self-end, .justify-self-center, etc..

Grid:

La bibliothèque fournit également un système de grid simple pour organiser vos mises en page. Voici les classes de grid disponibles :

  • .container: Définit une largeur maximale pour le contenu et le centre horizontalement.
  • .row: Définit une ligne dans la grille.
  • Classes de colonnes : .col-1 à .col-12.

Personnalisation:

Vous pouvez également personnaliser la bibliothèque en ajoutant vos propres styles ou en remplaçant les valeurs par défaut des variables CSS. Certaines fonctionnalités, telles que la scrollbar ou le système de grille, sont personnalisables via des variables dans les fichiers SASS du même nom.

Exemples d'utilisation:

Voici quelques exemples d'utilisation :

Centrer un élément et appliquer une marge en bas:

<p class="text-center mb-4"> Contenu </p>

Créer une grille simple et responsive:

<div class="container">
<div class="d-flex gap-4">
<div class="col-6">Contenu 1</div>
<div class="col-6">Contenu 2</div>
</div>
</div>
